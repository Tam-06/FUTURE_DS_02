# FUTURE_DS_02
Future Interns Data Science &amp; Analytics Tasks

## Project Overview

This project was completed as part of the Future Interns Data Science &
Analytics Program. The objective was to analyze customer churn using
Microsoft Power BI and identify the main factors influencing customer
retention.

## Tools Used

-   Microsoft Power BI
-   Power Query
-   DAX
-   Microsoft Excel

## Dashboard Features

-   KPI Cards
-   Churn by Contract
-   Churn by Internet Service
-   Churn by Payment Method
-   Customer Demographics
-   Interactive Slicers

## Key Insights

-   Churn rate is approximately 26.5%.
-   Month-to-month contracts have the highest churn.
-   Higher monthly charges are associated with higher churn.
-   Electronic check customers churn more frequently.
-   Long-term contracts improve customer retention.
